# dechroma.js

Chroma key compositing (e.g., green screen) for HTML canvas-manipulated video, using [ImageData](https://developer.mozilla.org/en-US/docs/Web/API/ImageData).

## Install

```sh
$ npm i dechroma
# or
$ yarn add dechroma
# or
$ pnpm add dechroma
```
